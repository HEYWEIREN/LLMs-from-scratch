# 为指令微调生成数据集（Generating Datasets for Instruction Finetuning）

该文件夹包含用于生成 instruction finetuning（指令微调）数据集的工具代码。

- [llama3-ollama.ipynb](llama3-ollama.ipynb)：使用 Llama 3 和 Ollama 创建 synthetic instruction finetuning dataset（合成指令微调数据集）的 notebook。

- [reflection-gpt4.ipynb](reflection-gpt4.ipynb)：实现基于 reflection-tuning（反思调优）的 instruction dataset refinement（指令数据集改进）步骤的 notebook。
